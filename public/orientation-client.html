GUIDAGE DU PRO

Comment souhaitez-vous orienter le prestataire vers votre domicile ?

📍 Localisation GPS (Maison)

Le pro utilise la position actuelle de votre maison comme point d'arrivée fixe.

💬 Guidage par Chat

Vous donnerez les instructions par message écrit (nom du bâtiment, étage, etc.).

📞 Guidage par Appel

Le prestataire vous appellera directement pour trouver l'entrée.
